# Swiss Data Generation Scripts

This project contains scripts to generate realistic Swiss customer data using Faker.js and real Swiss data sources.

Generated by IBM Bob

## Features

- ✅ Uses real Swiss lastnames extracted from official data
- ✅ **Weighted lastname selection based on actual frequency** (common names appear more often)
- ✅ Uses real Swiss postal codes and cities from AMTOVZ data
- ✅ Generates Swiss phone numbers (format: 0XX XXX XX XX)
- ✅ Generates Swiss mobile numbers (format: 07X XXX XX XX)
- ✅ Uses Faker.js with de_CH localization for other data
- ✅ Configurable number of records via command line parameter
- ✅ **Optimized for large datasets** - chunked writing with progress logging
- ✅ **High performance** - generates ~19,000 records/second

## Setup (First Time Only)

Before generating data, you need to extract the Swiss data:

### 1. Extract Unique Lastnames

Extracts all unique lastnames with frequency data from the source CSV:

```bash
node scripts/extract_lastnames.js
```

**Output:** `unique_lastnames.json` (207,848 unique Swiss lastnames with frequency data)

The script aggregates lastname frequencies across all Swiss cantons. For example:

- Most common: Müller (52,889 occurrences)
- Least common: Various rare names (3 occurrences)

### 2. Extract Swiss Locations

Extracts Swiss cities and postal codes from the AMTOVZ data:

```bash
node scripts/extract_swiss_locations.js
```

**Output:** `swiss_locations.json` (5,716 Swiss locations with postal codes)

## Data Generation

### Generate Swiss Customer Data

Generate a specified number of customer records:

```bash
node generate_data.js [number_of_records]
```

**Examples:**

```bash
# Generate 100 records (default)
node generate_data.js

# Generate 10 records
node generate_data.js 10

# Generate 1,000 records
node generate_data.js 1000

# Generate 10,000 records
node generate_data.js 10000

# Generate 2.5 million records (takes ~2-3 minutes)
node generate_data.js 2500000
```

**Output:** `output/generated_data_[number].csv`

### Performance

The script is optimized for large datasets:

- **Speed:** ~19,000 records/second
- **Memory efficient:** Writes data in chunks of 10,000 records
- **Progress tracking:** Real-time progress, speed, and ETA display

## Output Format

The generated CSV file contains the following columns:

- Nr. - Record number
- Anrede - Salutation (Herr, Frau, k.A.)
- Titel - Title (Dr., Prof., Prof. Dr., or empty)
- Vorname - First name (gender-appropriate)
- Nachname - Last name (from extracted Swiss lastnames, **weighted by frequency**)
- Geburtsdatum - Birth date (DD.MM.YYYY format, 1920-2010)
- Straße - Street name
- Hausnummer - House number
- Postleitzahl - Swiss postal code (4 digits)
- Stadt - Swiss city name
- Telefon - Swiss landline phone number
- Mobil - Swiss mobile phone number
- Telefax - Swiss fax number
- EMail - Email address
- Newsletter - Newsletter subscription (ja or empty)
- Eintragsdatum - Registration date (DD.MM.YYYY format, 1949-2025)

## Notes

- Some fields may be intentionally left empty to simulate real-world data
- Phone numbers follow Swiss formatting conventions
- All dates use Swiss date format (DD.MM.YYYY)
- Email domains use placeholder domains (.none)
- **Lastnames are selected using weighted random selection** - common names like "Müller", "Meier", "Schmid" appear more frequently, matching real Swiss demographics
